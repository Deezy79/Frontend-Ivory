mkdir command - create a new directory eg mkdir git_test touch command - create an empty file. Echo command - creates a file containing a text in it.eg echo "Hello world" > filename.txt cp command - to copy a file in the terminal. eg cp file1.txt test1.txt cat command - to display the contents of a file. eg cat file1.txt rm command - to delete a file eg rm file1.txt cd ~ command - root directory
